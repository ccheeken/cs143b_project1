## Compilation:

There is a Makefile to compile the program, so typing `make` 
in the command line (given you are in the folder, `project1`) 
will generate the executable, `p1`, for this project. Object 
files `manager.o` and `shell.o` are also created.

## How to run:

After compiling, you can run the program by typing
`./p1 <input_filename> <output_filename>`, into the 
command line where, `<input_filename>` is the name of 
the input file, and `<output_filename>` is the name of 
the file where the output of the program is stored. 

Example: `./p1 input.txt output.txt`
